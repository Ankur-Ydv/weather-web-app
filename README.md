# Weather-web-app
#### basic current weather checker WEB APP

## Demo
#### https://weather-web-app-pied.vercel.app/

## Installation Guide
### clone the project
```
  git clone git@github.com:Ankur-Ydv/weather-web-app.git
```
### go to the project directory
```
  cd weather-web-app
```
### Install dependencies
```
  npm install
```
### start the app
```
  npm run dev
```
